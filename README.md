# Игра «Камень–Ножницы–Бумага»

Классическая консольная игра против компьютера с подсчётом статистики.

## Описание

Проект реализует игру «Камень, ножницы, бумага» на языке Python.  
Вы играете против компьютера, который делает случайный ход. После каждого раунда выводится текущий счёт (победы, поражения, ничьи). Игра продолжается до тех пор, пока вы не введёте команду выхода.

Проект подходит для:
- изучения основ Python (условия, циклы, работа со словарями, модуль `random`);
- использования в качестве шаблона для более сложных игр (например, с расширенными правилами).

- ## Как играть

1. **Запустите скрипт** в любой среде Python (командная строка, терминал, IDE).
2. Введите один из допустимых ходов:
   - `камень`
   - `ножницы`
   - `бумага`
3. Компьютер мгновенно покажет свой выбор и объявит результат:
   - **Вы победили!** – если ваш ход бьёт ход компьютера.
   - **Компьютер победил!** – если ход компьютера бьёт ваш.
   - **Ничья!** – если ходы совпадают.
4. После каждого раунда отображается обновлённый счёт.
5. Чтобы завершить игру, введите `выход` или `exit` – тогда будет показан итоговый счёт за все сыгранные партии.

   ## Функциональность

- 🎮 **Интерактивный игровой цикл** – неограниченное количество раундов до команды выхода.
- 🤖 **Случайный ход компьютера** – используется модуль `random`.
- 📊 **Подсчёт статистики** – победы, поражения, ничьи выводятся после каждого раунда и в конце игры.
- 🧐 **Проверка ввода** – программа корректно реагирует на неверные команды и просит повторить ввод.
- 🔧 **Гибкие правила** – логика сравнения хранится в словаре, что позволяет легко расширить игру (например, добавить «ящерицу» и «Спока»).

## Требования

- **Python 3.6** или выше (протестировано на версиях 3.6–3.12).
- Стандартная библиотека Python – дополнительные пакеты не требуются.
